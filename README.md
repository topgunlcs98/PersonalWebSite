# PersonalWebSite
一个个人站点 Server: Node.js + Express.js / Client: React.js + React_Redux+ AntDesign
## 大概介绍
  1. React 前端框架
  2. Redux 数据管理，保持用户是否登陆状态
  3. Express/Node 简单后端连数据库（MongoDB）
  4. Antd UI库
## 实现功能
  1. 展示个人介绍，文章和简单简历
  2. 登陆，管理内容（修改简历，增删改博客）
